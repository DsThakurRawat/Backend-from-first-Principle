Inheritance in C++

This module covers C++ Inheritance models — Single, Multilevel, Multiple, and Hierarchical.

Definition: Inheritance allows derived child classes to inherit fields and methods from base parent classes, optimizing code reusability.

Inheritance Models

Inheritance Type Structure Syntax Example
Single Parent → Child class Car : public Vehicle
Multilevel Grandparent → Parent → Child class ElectricCar : public Car
Multiple ParentA + ParentB → Child class Duck : public Swimmable, public Flyable
Hierarchical Parent → ChildA & ChildB class Circle : public Shape

Access Rules

Using public inheritance preserves base access rules. Remember that private base variables are never directly accessible from inside derived child members. Use protected instead.

Example: Multiple Inheritance

class Duck : public Flyable, public Swimmable {
public:
    void action() {
        fly();
        swim();
    }
};
